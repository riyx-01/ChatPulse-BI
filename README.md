# ChatPulse: WhatsApp BI Dashboard

ChatPulse is a complete Business Intelligence dashboard for analyzing WhatsApp chat exports. Built with Streamlit, it offers deep insights into conversational patterns, sentiment, and user behaviors without requiring data to leave your machine.

## Features
- **Privacy-First**: No data is sent to external servers or APIs. All processing happens locally.
- **Deep Analytics**: See message volume trends, user activity, response times, and sentiment timelines.
- **Interactive Visualizations**: Powered by Plotly with a sleek dark theme.
- **PDF Reporting**: Export the entire dashboard as a professional PDF report.

## Setup Instructions

1. **Clone the repository** (or navigate to this directory):
   ```bash
   cd e:\my_projects\Chatpulse
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```

## Usage
1. Export a WhatsApp chat (without media) as a `.txt` file.
2. Open the ChatPulse dashboard in your browser.
3. Use the sidebar to upload your chat file.
4. Explore the interactive charts and metrics.
5. Click "Generate PDF Report" to download a summary.
